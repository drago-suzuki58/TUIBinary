# TUI Binary

My first C project.

A simple TUI Binary Editor written in C.  
Currently, it only supports reading binary files.

## Build

```sh
git clone https://github.com/drago-suzuki58/TUIBinary.git
cd TUIBinary
xmake
```

For more information about xmake, see [xmake.io](https://xmake.io/#/).

## Usage

```sh
./TUIBinary <filename>
```
